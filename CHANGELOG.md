## [1.0.0] - 2026-07-17

### Added

- Lossless single-frequency open/short shunt-stub solver with both canonical solutions.
- Interactive native SVG Smith chart with impedance, admittance, pointer, touch, and keyboard input.
- Shareable URL state, education, examples, SVG export, print worksheet, and offline reload.
- Cross-browser, mathematical-reference, property, accessibility, visual, performance, and security checks.
