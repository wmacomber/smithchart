# Mathematics

Phase 0 mathematical decisions remain authoritative in `SMITH_CHART_SPA_PLAN.md` until RF implementation documentation expands this file.

Locked subjects include complex-number conventions, impedance and admittance normalization, reflection conversion, toward-generator rotation, open and short shunt-stub equations, two-solution behavior, canonical half-wavelength intervals, tolerances, units, singularities, and active-load policy.

No RF equation is implemented during Phase 1. Phase 2 must add independently sourced reference cases, a formulation-independent verifier, deterministic tests, and fast-check invariants before exposing solver results.
