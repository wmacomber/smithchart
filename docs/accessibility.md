# Accessibility

Phase 1 shell uses header, main, section, footer, one page heading, logical heading order, visible focus, system color preference, responsive layout, and print-safe contrast.

Later phases must add keyboard-equivalent chart interaction, textual chart descriptions, accessible SVG names, associated input errors and units, live result announcements, non-color distinctions, touch targets, reduced motion, forced-colors support, zoom checks, and automated plus manual browser testing.
