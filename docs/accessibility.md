# Accessibility

Visible labels belong to every input. Invalid input retains committed value and exposes associated error. Load marker is keyboard-adjustable: arrows fine, Shift+arrows coarse, Enter commit, Escape cancel. Pointer interactions use large effective targets and touch-safe mapping.

Chart references synchronized prose. A/B use labels and line treatment, not color alone. Result changes use polite announcement; blocking errors use alerts. Focus remains visible. Motion follows user toggle and `prefers-reduced-motion`. Forced colors retain selected and load states.
